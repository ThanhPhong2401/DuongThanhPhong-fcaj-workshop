### Week 8 Objectives:

* Understand the Shared Responsibility Model and core AWS security services.
* Master identity and access management (IAM) and user authentication (Cognito).
* Learn centralized account management (Organization) and encryption key management (KMS).

| 2   | - Learn Module 05-03 & 05-04: Amazon Cognito (User/Identity Pools) and AWS Organization (account/billing management). | 06/06/2026   | 06/06/2026      | <https://youtu.be/N_vlJGAqZxo> |

### Week 8 Achievements:

* Gained clear understanding of customer versus AWS security roles in the shared responsibility model.
* Proficient in managing IAM users, roles, and policies for secure access control.
* Mastered user authentication for apps via Cognito and multi-account management with AWS Organization.
* Learned to centralize access via AWS Identity Center and manage data encryption keys using KMS.
