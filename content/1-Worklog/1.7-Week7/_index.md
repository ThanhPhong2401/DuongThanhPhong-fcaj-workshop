### Week 7 Objectives:

* Practice storage management, backup, and data security with Amazon S3.
* Learn the workflow for migrating virtual machines (VMs) from on-premise environments to AWS.
* Master techniques for importing/exporting virtual machines and managing cloud resources.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------ | --------------- | ----------------------------------------- |
| 1-2 | - Practice Lab 13: Set up S3 buckets, deploy infrastructure, configure Backup plans, set up notifications, perform data restores, and clean up resources. | 29/05/2026   | 30/05/2026      | <https://000013.awsstudygroup.com/2-prerequisite/2.1-creates3bucket/> |
| 3-5 | - Practice Lab 14: Export VMs from on-premise, upload to AWS, import into the platform, deploy instances from AMIs, and configure S3 bucket ACLs. | 31/05/2026   | 02/06/2026      | <https://000014.awsstudygroup.com/1-deploy-application-server/> |

### Week 7 Achievements:

* Mastered the workflow for setting up automated backups and restoring data on S3.
* Gained clear understanding of migrating, importing, and exporting VMs from on-premise environments to AWS.
* Proficient in managing S3 access permissions through bucket ACLs.
* Successfully deployed EC2 instances directly from migrated virtual machine files.
