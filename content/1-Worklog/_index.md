### Throughout my internship at the FCAJ AWS community, I documented my entire learning, research, and hands-on practice journey through this Worklog system.

#### The internship program was structured and completed over **12 weeks**. Below is a summary of the key tasks and achievements I accomplished each week:

**Week 1:** [FCAJ program kickoff, AWS account registration, and completing 5 core AWS service tasks](1.1-week1/)

**Week 2:** [Learning core VPC networking, configuring subnets, and managing Security Groups and NACLs](1.2-week2/)

**Week 3:** [Studying advanced VPC Peering and Transit Gateway connectivity and setting up a Bastion Host](1.3-week3/)

**Week 4:** [Deploying AWS Transit Gateway, exploring Load Balancers, and working with EBS and EFS storage](1.4-week4/)

**Week 5:** [Exploring EC2 Auto Scaling, configuring cross-region VPC Peering, and implementing Storage Gateway](1.5-week5/)

**Week 6:** [Deep dive into Amazon S3 architecture, configuring static website hosting, and exploring Snow Family and backup solutions](1.6-week6/)

**Week 7:** [Hands-on with S3 Backup plans, practicing VM import/export, and migrating virtual machines to AWS](1.7-week7/)

**Week 8:** [Studying the shared responsibility model, Amazon Cognito, AWS Organization, and KMS key management](1.8-week8/)

**Week 9:** [Learning foundational database concepts, completing security labs, and evaluating resources with AWS Security Hub](1.9-week9/)

**Week 10:** [Deep dive into Amazon RDS and Aurora, practicing DB administration labs, and drafting the group project architecture](1.10-week10/)

**Week 11:** [Exploring Amazon Redshift and ElastiCache solutions, executing advanced DB labs, and publishing technical blogs for the community](1.11-week11/)

**Week 12:** [Finalizing AWS architecture, publishing blogs on S3 Access Logs and FSx for Oracle DB, and conducting the group project demo](1.12-week12/)
